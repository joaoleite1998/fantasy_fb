This is a basic fantasy football app which collects data directly from the football database website. After creating your login, you can draft quarterbacks, running backs, wide receivers, tight ends and kickers to create an ultimate roster. You are also able to view each player's current fantasy points, as well as his fantasy statistics from previous seasons. There is no deadline to drafting and dropping, but if another user drafts a player before you, then you can no longer draft that player, unless the player is then dropped. Statistics can be updated as new NFL games take place in the current season by clicking on the update button on the home page.
